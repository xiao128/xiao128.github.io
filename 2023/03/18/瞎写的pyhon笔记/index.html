python中的各种符号以及代码的基本内涵

Python有很多种符号，它们可以分为以下几类

- 算术运算符：用来进行加、减、乘、除等数学运算，例如+、-、*、/等。

+：加法，例如2 + 3 = 5

-：减法，例如5 - 2 = 3

*：乘法，例如2 * 3 = 6

/：除法，例如6 / 3 = 2

//：整除，例如7 // 3 = 2

%：取余，例如7 % 3 = 1

**：幂，例如2 ** 3 = 8

- 赋值运算符：用来给变量赋值，例如=、+=、-=等。

=：简单赋值，例如x = 5

+=：加法赋值，例如x += 2相当于x = x + 2

-=：减法赋值，例如x -= 2相当于x = x - 2

*=：乘法赋值，例如x *= 2相当于x = x * 2

/=：除法赋值，例如x /= 2相当于x = x / 2

//=：整除赋值，例如x //= 2相当于x = x // 2

%=：取余赋值，例如x %= 2相当于x = x % 2

**=：幂赋值，例如x **= 2相当于x = x ** 2

- 比较运算符：用来比较两个值的大小或相等性，例如==、!=、>、<等。

==：相等，例如2 == 2返回True

!=：不等，例如2 != 3返回True

：大于，例如3 > 2返回True

<：小于，例如2 < 3返回True

>=：大于等于，例如3 >= 2返回True

<=：小于等于，例如2 <= 3返回True

- 逻辑运算符：用来进行布尔逻辑运算，例如and、or、not等。

and：逻辑与，例如True and False返回False

or：逻辑或，例如True or False返回True

not：逻辑非，例如not True返回False

- 位运算符：用来进行二进制位的操作，例如&、|、^等。

二进制的光我啥事，看啥看，又不考，下一个

- 成员运算符：用来检查一个值是否在一个序列或集合中，例如in和not in。

in：判断一个值是否在一个序列中，例如3 in [1, 2, 3]返回True

not in：判断一个值是否不在一个序列中，例如4 not in [1, 2, 3]返回True

- 身份运算符：用来检查两个对象是否是同一个对象，例如is和is not。

is：判断两个对象是否是同一个对象，例如a is b返回True，如果a和b引用的是同一个对象

is not：判断两个对象是否不是同一个对象，例如a is not b返回True，如果a和b引用的不是同一个对象

- 运算优先级：用来确定不同的符号在表达式中的执行顺序，例如()表示括号内的内容优先计算。

根据Python官方文档 ，Python中的运算优先级从高到低如下：

()：圆括号

**：幂

+x, -x, ~x：正号，负号，按位取反

*, /, %, //：乘法，除法，取余，整除

+, -：加法，减法

<<, >>：左移，右移

&：按位与

^：按位异或

|：按位或

<, <=, >, >=: 比较大小

<=>: 比较大小和相等（Python 3.10新增,信息会考不考>别看了）

==, !=: 比较相等和不等

is, is not: 比较身份

in, not in: 比较成员关系

not: 逻辑非

and: 逻辑与

or: 逻辑或

- Python还有一些特殊的符号，它们表示一些内置的函数或模块23。例如：

print()：打印输出字符串或其他对象

sqrt()：计算一个数的平方根

pow()：计算一个数的幂

#：用于表示注释，Python解释器会忽略以#开头的行。

\：用于表示转义字符，例如\n表示换行，\t表示制表符，\\表示反斜杠本身2。

@：用于表示装饰器（decorator），它是一个函数，可以在不修改原函数代码的情况下给原函数增加一些功能

一般来说，Python中缩进的规则有以下几点：

对于类定义、函数定义、流程控制语句、异常处理语句等，行尾的冒号和下一行的缩进，表示下一个代码块的开始，而缩进的结束则表示此代码块的结束。

相同逻辑层（同一个代码块）保持相同的缩进量。

缩进量（字符个数）不限，但通常情况下都是采用4个空格长度作为一个缩进量（一个Tab键就表示4个空格）。

不建议混用空格和制表符（tab符）标记缩进，因为这会影响代码阅读，增加维护及调试的困难。
