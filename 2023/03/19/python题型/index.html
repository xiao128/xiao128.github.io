计算5~51之和

核心考察点：range函数的使用方法（即区间函数）

range(终值)：默认起始值为0，步长为1，返回从0到终值（不包括终值）的整数列表。例如：range(5) 等价于 [0, 1, 2, 3, 4]

range(起始值, 终值)：默认步长为1，返回从起始值到终值（不包括终值）的整数列表。例如：range(1, 5) 等价于 [1, 2, 3, 4]

range(起始值, 终值, 步长)：返回从起始值到终值（不包括终值）的整数列表，按照指定的步长递增或递减。例如：range(1, 10, 2) 等价于 [1, 3, 5 ,7 ,9]

输入一个正整数n，输出从1到n的所有奇数。

方法一

意义：代码中首先输入一个正整数n，然后初始化变量i为1。在while循环中，如果i是奇数，则输出i，然后将i加1，进入下一轮循环。当i大于n时，循环结束。这样就可以输出从1到n的所有奇数。

考点：while指令的运用

while是Python中的一种循环结构，可以重复执行一段代码块，直到满足指定条件时停止循环。通常用于处理需要重复执行的任务，如输入验证、数据处理、游戏循环等。

while语句的一般语法如下：
while condition:

其中，condition是一个表达式，通常包含一个或多个变量和运算符，用于检查循环是否继续执行。在每次循环开始时，都会先检查condition的值，如果为True，则执行代码块中的语句；如果为False，则退出循环。

chr(ord(‘a’)+1)的值为？（选择题）

在这个表达式中，ord(‘a’) 将字符 ‘a’ 转换成其对应的 ASCII 码值（整数），而 chr() 函数则将整数转换为其对应的字符。

因此，ord(‘a’)+1 计算出的是 ‘a’ 后面的一个字母 ‘b’ 的 ASCII 码值然后 chr() 函数将该整数转换成对应的字符 ‘b’。

这样的计算顺序是正确的，因为先计算 ord(‘a’) 保证了对应的 ASCII 码值正确，再通过 chr() 函数将该整数转换成对应的字符。
