  • 建站第一天

    建站第一天，为了搞这个博客博主差点去世

    （菜鸡博主）（至少因为这个作业写不完了，死掉了，晚上要被大文真实了😥）

    7.1 发布文章

    进入博客所在目录，右键打开 Git Bash Here，创建博文：

    hexo new “My New Post”
    然后 source 文件夹中会出现一个 My New Post.md 文件，就可以使用 Markdown 编辑器在该文件中撰写文章了。

    写完后运行下面代码将文章渲染并部署到 GitHub Pages 上完成发布。以后每次发布文章都是这两条命令。

    hexo g # 生成页面
    hexo d # 部署发布
    也可以不使用命令自己创建 .md 文件，只需在文件开头手动加入如下格式 Front-matter 即可，写完后运行 hexo g 和 hexo d 发布。看到这已经该无聊的睡着了😴
